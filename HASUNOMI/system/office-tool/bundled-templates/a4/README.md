# Supplied A4 template

This is the one shared visual A4 template supplied with the pack. Ordinary A4
documents, meeting minutes, and meeting resumes use its components; they are
not separate visual templates.

- Output: editable ODT by default.
- Page: A4 portrait, 13/15/13/15 mm margins.
- Typography: Noto Sans CJK JP for the Japanese support profile.
- Components: title, optional lead, three semantic heading levels, paragraphs,
  lists, normal tables, `metadata` tables, callouts, optional handwriting
  areas, and an optional footer.
- Tables: visible borders, readable cell padding, compact table paragraphs,
  and content-aware column widths.
- Meeting-minute and meeting-resume examples are references only. Select,
  omit, reorder, or add components to fit the purpose of the actual meeting.
- Content data contains structure only. It does not specify fonts or layout.

`example.json` is a general A4 example. `meeting-minutes-example.json` and
`meeting-resume-example.json` demonstrate possible component selections for
meetings; neither is a fixed form.

New user templates belong in protected `TEMPLATES/`, not in this replaceable
bundled-template directory.
