# MEMORY Index

This file is the search entrypoint for reusable preferences and lessons.

## How to use this index

- Search this file before opening `notes/`.
- Keep entries short and searchable.
- Link a detailed note only when the index entry is not enough.
- Update or replace old entries instead of creating duplicates.
- Keep current project state in `PROJECTS/.../MASTER.md`, not here.
